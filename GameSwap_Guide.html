GameSwap
Guide d'utilisation — partage de jeux en réseau local
Version 1.0  ·  Avril 2026

Qu'est-ce que GameSwap ?

GameSwap permet à plusieurs joueurs d'un même réseau (box internet ou routeur commun) de partager et télécharger des jeux entre eux, sans clé USB ni disque dur externe. Chaque joueur met ses jeux à disposition sur le réseau, les autres les voient et peuvent les récupérer en quelques clics.

💡 En résumé Vous partagez vos jeux → vos amis les voient → ils téléchargent ce qui les intéresse → ils jouent directement.

⚠️ Important GameSwap doit toujours être lancé via Start-GameSwap.bat (clic droit → Exécuter en tant qu'administrateur si demandé).

Premier lancement

La première fois, un assistant vous guide en deux étapes :

  1. Double-cliquez sur Start-GameSwap.bat. Acceptez la demande d'élévation si Windows vous la propose.
  2. Choisissez le dossier d'installation (là où vos jeux seront stockés). Exemple : D:\ ou C:\GameSwap.
  3. Entrez votre nom de joueur (lettres et chiffres uniquement, 20 caractères max). C'est le nom que les autres verront sur le réseau.
  4. Cliquez sur Terminer. GameSwap configure tout automatiquement.
✅ Après le premier lancement Un partage réseau est créé automatiquement sur votre PC. Les autres joueurs pourront voir vos jeux depuis leur propre GameSwap.

L'interface principale

L'application comporte trois onglets accessibles en haut de la fenêtre :

  • 🎮 Mes Jeux — vos jeux installés sur votre PC
  • 🌐 Réseau — les jeux disponibles chez les autres joueurs
  • ⚙ Paramétrage — vos réglages et outils

Onglet — Mes Jeux

Cet onglet affiche tous les jeux présents dans votre dossier GameSwap.

Actions disponibles

  • ⬇ Installer Lance l'installation du jeu (configuration automatique).
  • ▶ Jouer Monte le jeu et le lance directement.
  • ► Serveur Lance un serveur dédié (si le jeu le supporte).
  • Désinstaller Retire la configuration du jeu (le fichier reste).
  • Supprimer Supprime définitivement le fichier du jeu.

Panneau de droite

En sélectionnant un jeu dans la liste, le panneau droit affiche sa vignette, son année de sortie, le nombre de joueurs max, sa description et ses instructions.

💡 Astuce Un jeu affiché en jaune n'est pas encore installé — vous pouvez quand même le mettre à disposition sur le réseau pour que les autres le téléchargent.

Onglet — Réseau

Cet onglet permet de voir et télécharger les jeux des autres joueurs.

Scanner le réseau

  1. Cliquez sur 🔍 Scanner le réseau. La recherche dure quelques secondes.
  2. Les joueurs trouvés apparaissent dans la liste de gauche. Cliquez sur un joueur pour voir ses jeux.
  3. Sélectionnez un jeu dans la liste du bas pour voir sa description à droite.

Télécharger un jeu

  1. Sélectionnez le jeu souhaité puis cliquez sur ⬇ Télécharger ce jeu.
  2. La progression s'affiche en bas (pourcentage, temps écoulé, temps restant).
  3. Une fois téléchargé, le jeu apparaît automatiquement dans l'onglet Mes Jeux.
✅ File d'attente automatique Si le joueur distant a atteint son nombre maximum de téléchargements simultanés, votre téléchargement se met en attente et démarre automatiquement dès qu'une place se libère.
⚠️ Annuler Utilisez ■ Stopper pour annuler un téléchargement en cours, ou Annuler l'attente pour quitter la file d'attente.

Onglet — Paramétrage

Nom du joueur

Modifiez votre nom visible sur le réseau puis cliquez sur Appliquer.

Dossier des jeux

Changez l'emplacement de votre bibliothèque. Vous pourrez déplacer vos jeux existants vers le nouveau dossier.

Téléchargements simultanés

Définissez combien de joueurs peuvent télécharger chez vous en même temps (par défaut : 3). Cliquez sur Appliquer pour enregistrer.

Redistribuables système

Ces boutons installent les composants nécessaires à de nombreux jeux :

  • Visual C++ (toutes versions) — librairies requises par la plupart des jeux PC (2005 à 2022).
  • DirectX June 2010 — composants DirectX 9 nécessaires aux jeux anciens.
💡 À installer une seule fois Si certains jeux ne démarrent pas, commencez par installer ces deux redistribuables.

Questions fréquentes

Les autres joueurs ne voient pas mon partage

  • Vérifiez que vous êtes bien sur le même réseau (même box/routeur).
  • Vérifiez que le pare-feu Windows n'est pas en train de bloquer le partage de fichiers.
  • Relancez GameSwap — le partage est recréé automatiquement au démarrage.

Un jeu ne se lance pas après installation

  • Installez les redistribuables Visual C++ et DirectX depuis l'onglet Paramétrage.
  • Consultez les instructions du jeu dans le panneau droit de l'onglet Mes Jeux.

Le téléchargement est bloqué en attente

  • Le joueur source a atteint sa limite de téléchargements simultanés.
  • GameSwap reessaie automatiquement toutes les 30 secondes.
  • Vous pouvez cliquer sur Annuler l'attente pour abandonner.

Où se trouvent les logs en cas de problème ?

  • Dans le dossier logs\ à la racine du dossier GameSwap.
  • Le fichier du jour est nommé GameSwap_AAAAMMJJ.log.
